FAQ

Questions, answered

Is my data sent anywhere?

No. There is no backend and no analytics, so your report is never uploaded anywhere. Your entries are saved only to this browser's local storage, and clearing them removes them permanently. The one and only network request the page can make is fetching the PDF-rendering library from a CDN, and that happens only if you click Download PDF — the library is fetched, your text is not sent. Everything else, including the Word export, works fully offline.

Does it use AI?

Not today. It's a deterministic formatter: it takes your bullets, structures them, and writes a summary line from your item counts and mood. That means it's instant, free, and works offline. An optional AI polish pass is on the roadmap.

Can I use it for standups or weekly reports?

Yes. The same structure — done, in progress, blocked, next — is exactly what a standup or weekly wants. Change the date field and paste a week's worth of bullets.

What's the “% done” thing in the In Progress box?

Purely optional. Write API refactor — 60% and the report renders it as a progress item. Skip it and the line stays a plain bullet.

Does it work on my phone?

Yes — the panes stack and every button stays tappable. Type on the train, copy, paste into Slack, done.
